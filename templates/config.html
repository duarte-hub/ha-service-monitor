Setup instructions
  1. Go to console.cloud.google.com → create or select a project
  2. Enable Google Drive API
  3. APIs & Services → Credentials → Create Credentials → OAuth client ID
  4. Application type: TVs and Limited Input Devices
  5. Copy the Client ID and Client Secret below and save
  6. Click Connect Google Drive — a code will appear
  7. Open the URL shown, enter the code, and grant access
